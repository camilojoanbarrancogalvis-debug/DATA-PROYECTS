WA_sales_products es un base de datos proveninente de IBM. Esta contiene datos ficticos que simulan una empresa de venta de articulos a nivel global. El objetivo del proyecto es realizar analisis que permitan describir la situacion de la empresa y consolidar un dashboard que muestre KPI's como los ingresos, costos y la utilidad bruta.
